# Commotion Skills

Claude Skills and an MCP adapter for operating Commotion voice agents through Claude.

## Overview

This project provides a set of [Claude Skills](https://docs.claude.com/en/docs/claude-code/skills)
and a supporting MCP server for building, testing, and improving voice agents — driven
through Claude as a single iterative loop:

```
Generate scenarios → Simulate → Evaluate → Diagnose → Create/Update agent → repeat
```

The skills orchestrate the loop; the MCP adapter exposes the underlying platform
capabilities as tools. All agent write-back is human-approved.
